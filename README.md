# Financial Fraud Detection System

A Machine Learning-based Financial Fraud Detection System built using Random Forest Classifier and Streamlit.

## Project Overview

This project predicts whether a financial transaction is fraudulent or legitimate using Machine Learning techniques.

The model is trained on transaction data using features like:

- Transaction type
- Amount
- Balance details
- Transaction step

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Random Forest Classifier
- Machine Learning

## Features

- Fraud transaction prediction
- Interactive Streamlit web application
- Real-world transaction dataset
- Classification-based ML model
- Clean project structure

## Machine Learning Workflow

1. Data Cleaning
2. Label Encoding
3. Feature Selection
4. Train-Test Split
5. Model Training
6. Prediction
7. Evaluation

## Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Streamlit app:

```bash
streamlit run app.py
```
